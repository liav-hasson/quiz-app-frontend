# Scripts

Utility scripts for frontend build and deployment.

---
## Scripts

| Script | Description |
|--------|-------------|
| `compute_next_version.sh` | Calculate next semantic version from git tags |
| `update-gitops.sh` | Update GitOps repository with new image tag |

---
## Usage

Scripts are called by the Jenkins pipeline during CD stages.
